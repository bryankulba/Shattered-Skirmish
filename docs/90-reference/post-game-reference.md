# SHATTERED – Post-Game Reference Sheet

---

## Post-Battle Sequence

Resolve in this order after every battle.

| Step | Action |
|---|---|
| 1 | **Determine Outcome** — Resolve objectives. Routed warband loses. |
| 2 | **Injury Resolution** — Roll on Injury Table for every model taken Out of Action. |
| 3 | **Secure Resource Caches** — Caches held by surviving/withdrawing models are Secured. Others are lost. |
| 4 | **Resource Conversion** — Per Secured Cache: roll 1D3 → gain that many RP. Gain 1 Salvage. RP from this step cannot be spent this sequence. |
| 5 | **Repair & Refit** — Spend Salvage now. Unspent Salvage is retained. |
| 6 | **Exploration** — Roll Exploration dice. Resolve results. |
| 7 | **Recruitment** — If a Recruitment Opportunity is available, spend 5 RP and roll on the New Recruit Table, or decline. |
| 8 | **Experience & Advancement** — Award XP. Resolve advancements. |
| 9 | **Update Roster** — Record everything. Prepare for next battle. |

---

## Injury Resolution

Roll for every model taken Out of Action. Apply results immediately.

### Hero Injury Table — roll 2D6

| 2D6 | Result |
|---|---|
| 2 | **Dead** — Remove from roster. |
| 3 | **Captured/Missing** — Miss next battle. Returns with *Armour Compromised* if warband controls field next game; otherwise removed. |
| 4 | **Multiple Trauma** — Roll twice (re-roll Dead). Apply both. |
| 5 | **Armour Compromised** — –1 Armour Save until repaired (2 Salvage). |
| 6 | **Weapon Damaged** — One weapon gains *Damaged* until repaired (1 Salvage). |
| 7 | **Full Recovery** — No lasting effects. |
| 8 | **Concussion** — –1 Initiative next battle only. |
| 9 | **Crippling Wound** — –1 Movement permanently. |
| 10 | **Oath of Moment** — Choose a named foe; +1 Ld on Rout tests and re-roll one failed To Hit per battle against them. |
| 11 | **Bionic Replacement Needed** — –1 WS until repaired or stabilised. |
| 12 | **Hardened by Fire** — +1 XP immediately. Re-roll the next Advance roll. |

### Henchman Injury Table — roll 2D6

| 2D6 | Result |
|---|---|
| 2 | **Dead** — Remove from roster. |
| 3 | **Lost in the Ruins** — Miss next battle. Returns with *No Weaponry* if warband wins next game; otherwise removed. |
| 4 | **Deep Wounds** — Miss next battle. Returns with *Armour Compromised*. |

## Salvage Spending

Salvage may only be spent during Step 5: Repair & Refit.

| Action | Cost |
|---|---|
| Repair Armour Compromised | 2 Salvage |
| Repair Damaged Weapon | 1 Salvage |
| Re-equip Weaponless Recruit | 1 Salvage |
| Stabilise Injury (re-roll one non-permanent result; must accept second) | 1 Salvage |

Salvage cannot improve characteristics, purchase weapons, or recruit models.

## Exploration

Resolved during Step 6.

**Roll Exploration dice:**
- 1D6 per surviving Hero.
- +1D6 if the warband controlled the field.
- If more than 3 dice are rolled, retain only the three highest.

**Resolving results:**
1. Check retained dice for duplicates.
2. If duplicates exist → use Duplicate Table (overrides Total Table).
3. If all unique → add the three dice and use Total Table.
4. If any retained die shows a natural 6 → gain 1 Recruitment Opportunity (one per sequence).

## Recruitment

Resolved during Step 7. Requires a Recruitment Opportunity and an open warband slot.

- Pay the full RP cost of the model type.
- Roll 2D6 on the Recruitment Table.
- Roll 2D6 on the Allegiance Table.
- Apply all results immediately.

Recruits begin with 0 XP. Must follow all Warband Creation equipment restrictions.

## Experience & Advancement

### XP Awards (per battle)

| Condition | XP |
|---|---|
| Survive the battle | +1 |
| Take an enemy Out of Action | +1 |
| Scenario contribution | +1 (max once per battle) |
| Warband controls the field (Veteran Sergeant only) | +1 |

Maximum +3 XP per model per battle.

### Advancement Roll

When a model reaches an advancement threshold, roll 3D6 on the Advancement Table.

- Results of 3, 12, or 18 → roll 1D6 on the Characteristic Table instead.
- If a result cannot be applied, re-roll.
- Each characteristic increase may only be gained once per model.

| 3D6 | Result |
|---|---|

| 9 | **Brutal Charge** — +1 Strength in the first round of a charge. |
| 10 | **Heart of the Legion** — If this model did not move this activation, +1 FP when firing. |
| 11 | **Precision Fire** — Does not need to target the closest model when firing. |
| 13 | **Battlefield Calculus** — Once per battle, after both sides roll for activation priority, may re-roll your result. |
| 14 | **Fearless Aspect** — Causes Fear. |
| 15 | **Executioner** — If this model takes an enemy Out of Action in melee, gain +1 XP (max once per battle). |
| 16 | **Battlefield Predator** — +1 Attack in the first round of combat. |
